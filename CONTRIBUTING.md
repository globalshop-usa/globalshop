# Contributing to GlobalShop Documentation

Thank you for your interest in improving GlobalShop documentation.

## About This Repository

This repository contains public documentation for the GlobalShop e-commerce platform operated by C&A Marketing Inc. It is not an open-source software project.

## What You Can Contribute

- Documentation corrections
- Typo fixes
- Clarity improvements
- Translation suggestions
- FAQ additions

## How to Contribute

### Reporting Issues

1. Check existing issues first
2. Open a new issue with clear description
3. Include relevant links or screenshots

### Suggesting Changes

1. Fork the repository
2. Create a branch for your changes
3. Make your edits
4. Submit a pull request with description

### Style Guidelines

- Use clear, concise language
- Follow existing formatting
- Include relevant links
- Update "Last Updated" dates

## What We Cannot Accept

- Code contributions (platform is proprietary)
- Feature requests (use gshop_tiktok@outlook.com)
- Business inquiries (use gshop_tiktok@outlook.com)
- Legal questions (use gshop_tiktok@outlook.com)

## Code of Conduct

- Be respectful and professional
- Stay on topic
- No spam or self-promotion
- No confidential information

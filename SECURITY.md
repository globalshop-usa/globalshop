# Security Policy

## Reporting a Vulnerability

C&A Marketing Inc takes security seriously. If you discover a security vulnerability in GlobalShop, please report it responsibly.

### Contact

**Email**: gshop_tiktok@outlook.com

**PGP Key**: Available upon request

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested remediation

### Response Timeline

| Stage | Timeframe |
|-------|-----------|
| Acknowledgment | 24 hours |
| Initial Assessment | 72 hours |
| Status Update | 7 days |
| Resolution Target | 30-90 days |

### Scope

**In Scope:**
- g-shops.net and subdomains
- GlobalShop mobile applications
- GlobalShop APIs
- Seller Center portal

**Out of Scope:**
- Third-party services
- Social engineering attacks
- Physical security
- Denial of service attacks

### Safe Harbor

We will not pursue legal action against researchers who:
- Act in good faith
- Avoid privacy violations
- Do not destroy data
- Report promptly and privately
- Allow reasonable time to remediate

### Recognition

We acknowledge security researchers who help improve our platform (with permission).
